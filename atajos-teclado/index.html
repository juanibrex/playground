Atajos de teclado

Atajos de teclados creados o editados por mí. Y otros útiles del sistema

  • Ctrl + B Abre y cierra (toggle) panel de archivos
  • Ctrl + Ñ Abre y cierra (toggle) panel de actividad (iconos)
  • Ctrl + ' Abre y cierra (toggle) la Terminal integrada
  • Ctrl + ¿ Comentario de linea (Antes Ctrl + Shift + 7)
  • Ctrl + Shift + ¿ Comentario de bloque (Antes Ctrl + Shift + A)
  • Ctrl + Alt + downArrow Duplica la linea seleccionada*
    • Ctrl + Alt + Shift + downArrow Copia una linea hacia abajo
    • Ctrl + Alt + Shift + upArrow Copia una linea hacia arriba
  • Ctrl + Shift + k Borrar linea
  • Alt + up/down Mueve elementos hacia arriba o hacia abajo.
  • Alt + A envuelve o contiene (wrapper) un elemento u texto seleccionado con otro dado.
